# sms-send-service
Simple sms send service

## USE
### Install packages
```
$ yarn
```
### Start server
```
$ yarn start
```
---
## Endpoints
### Send SMS
URL
```
POST {domain}/send
```
BODY
```
{
    "phoneNumber": "XXXXXXXX",
    "message": "TEST"
}
```
